# slay-the-spire

Slay the Spire 2 Wiki — "Spire Tome", a static deck-building reference wiki.

## Pages

| File | Page |
| --- | --- |
| `index.html` | Home / Library landing page (Compendium overview) |
| `characters.html` | Character roster — The Champions of Aethelgard |
| `necrobinder-cards.html` | Necrobinder Card Library — The Necrobinder's Grimoire |
| `relics.html` | Relic Compendium (rarity filters and synergy tags) |
| `symbols.html` | Compendium of Symbols (iconography reference) |
| `glossary.html` | Glossary of Terms (keywords, stats, potions) |
| `strategies.html` | Tactical Compendium — run archetypes and boss strategy |
| `poison-strategy.html` | The Serpent's Coil — Act 1 poison strategy guide |
| `bestiary.html` | The Spire Bestiary — normal, elite, and boss enemies |
| `map-events.html` | Map & Events — pathing, merchant rates, encounters |
| `merchant.html` | The Merchant's Bazaar — shop services and pricing |
| `meta-progression.html` | Meta-Progression & Ascension — unlocks and run records |
| `hud.html` | HUD & Exploration Mechanics — the Explorer's Interface |

All pages are self-contained static HTML styled with Tailwind (via CDN), EB Garamond / Hanken Grotesk fonts, and Material Symbols icons. Top/side navigation links are wired between pages by their visible labels (e.g. Library, Cards, Relics, Bestiary, Map, Merchant). A few labels with no dedicated page yet (Settings, Lore) remain placeholder anchors.

To view, open any `.html` file in a browser.
